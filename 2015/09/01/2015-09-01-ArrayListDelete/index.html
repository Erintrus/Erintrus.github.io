比较两段代码的不同

示例一：

  Iterator<ShowInfo> iterator = showList.iterator();
if (iterator.hasNext()) {
      ShowInfo delShow = iterator.next();
       if (delShow.isCheck() == true) {
           iterator.remove();
       }
   }

示例二

for (Iterator<ShowInfo> delList = showList.iterator(); delList.hasNext(); ) {
    ShowInfo delInfo = delList.next();
    if (delInfo.isCheck() == true) {
        delList.remove();
    }
}
